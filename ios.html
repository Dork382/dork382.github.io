Hystory

In 2005 Steve Jobs tried to create an operative system for our mobile-phones , His idea was the one to carry the experience of Mac (MacOS) on a mobile phone , when he proposed is idea, he had to discuss with the entire Apple team, and after many discussions, they created the IPhone OS, which is commonly known now as the iOS system, which got placed in the market on 9th January 2007. In the beginning, the iOS could only support softwares developed by the Apple, because Steve Jobs said that he didn’ t want the iOS to support softwares created by other agencies, just to protect the system. In October 2007, Jobs changed idea, and he gaves the chance to create a native SDK for the Iphone. The Apple store was created on the 10th July 2008 with 500 softwares/applications, available. Actually in 2020 there are 1 milion softwares/applications available on the Apple store. In June 2010 , Apple renamed iPhone OS in "iOS". The "iOS" brand name got used by Cisco for ten years, until Cisco decided to gave the name to Apple. In October 2016, Apple created the first iOS Developer Academy placed in Naples ,inside the campus of Naples University.





Pros

The iOS system has got many advantages, that we are going to describe:
•The APPs are always optimized, because the model that iOS use aren’t that much. And for the APP’s developers is really simple to creat adapted models, that’ s becuase we never have resolution problems.
•It’ s really safe, because the APPs that you can download are the only ones placed in the APP store which always checks them to see if they ve got some malwares inside the programming code.
•The updates are very fast and every year a new update for the system gets released. And the users can update it instantly and fast.
•It’ s really simple to use it because the APP’ s are always optimized for the users which are not experts.
•It’ s part of an ecosystem, this means that if you start a task on your iPhone, you can complete it on an iPad or a MAC, because they re compatible to each other.





Cons

1. Apple Ecosystem
The Apple ecosystem is both a boon and a curse. Once you get in, it's really hard to get out. Because there are so many differences between android and ios that people prefer not to change it.
2. Interactive
Unfortunaly there’s a limited interactivity in the homescreen because we can’ t use that we would like to have like, having buttons for (play/pause, weather location switch buttons).
3.Cons of Developing iOS Apps

•Expensive - The Xcode IDE can only be used on Macs. This significantly increases initial investment before you even start developing it.
•Stringent publishing - Apps which developers create must pass an exam that has high standards. Apple and iOS are quite restrictive when an app doesn't meet the criteria, and many apps get rejected.
•Customization - Because of the standardization guidelines, many apps lose initial creativity and become "like the rest." This lack of customization turns away many developers.
4.No expandable storage and battery
The amount of storage that you have when you purchase an Apple phone will never change. Because the operating system does not allow you to add additional memory and change the battery.
5.No expandable storage
iOS and the Applications which run within it are extremely limited with how they handle information. This may not be a problem for you as an user but if you run into that wall the restrictions will become very apparent.





Feature

Siri is a virtual vocal helper of iOS which permits to use the operative system with a vocal system, and because it is an “AI”, it’ s always updated ì iCloud is an iOS service used as a backing storage placed in a cloud server. The design is common to every iOS system. iOS only permits the use of hardwares authorized by Apple. iOS use a programming language called Swift, created by the Apple, so it’ s an original programming language.





Comparison

So, if we have to comparize the iOS system with the Android system, we can say that differently from Android, ios is way more safe, because the iOS system can' t be modified or corrupted, and the apps are always checked before they get exposed in the App Store, the programming language which is called Swift is faster than android, while Android uses javascript which is really slow, the UI design has got a user friendly interface which is really simple to use, the updates are always on point, the iOS system gets updated every year, and also Android but iOS always includes new functionalities, finally but not less important is the Apple' s ecosystem which permits to every Apple devices to shares data with another Apple device, with a fast execution.
